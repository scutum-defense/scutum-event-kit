# Changelog

## [0.1.0] - 2026-04-01

### Added
- Initial release of `@scutum/event-kit`
- `EventBus` with subscribe, publish, and query capabilities
- `TypedEventBus` for type-safe event emission and handling
- `EventEnvelope` schema with correlation and causation tracking
- `ValidationMiddleware` for event type validation
- `LoggingMiddleware` for event logging
- `JsonSerializer` for event serialization/deserialization
- Scutum Event Protocol v1.0.0 with 12 core event types
- Comprehensive test suite with Vitest
- TypeScript strict mode with full type declarations
